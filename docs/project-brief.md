# RipCrypt FoundryVTT System — Project Brief

We are building a custom Foundry VTT system for RipCrypt.

The first milestone is a functional character sheet, not a full automation suite. The sheet should allow a user to create a character, enter all core stats/resources, manage inventory or abilities, and perform the core rolls needed during play.

## Priorities

1. Foundry v14-compatible package scaffold.
2. Actor and Item data models.
3. Character actor sheet.
4. Basic item sheets.
5. Roll mechanics.
6. Localization.
7. Basic CSS layout.
8. Later: compendia, automation, polish, NPC sheets, drag/drop improvements.

## Non-goals for Initial Milestone

- No complex build pipeline.
- No full compendium data entry unless requested.
- No speculative mechanics not confirmed from the rulebook.
- No support for old Foundry versions.
